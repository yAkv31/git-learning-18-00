Hello git

Learning git and we are super!

  • git init - Команда которая инициализирует git-репозиторий!
  • git config user.email, git config user.name - это наше имя и почта по которой производится момент сохранения
  • git config gui.encoding utf-8 - поддержка кириллицы
  • git add - это добавление файлов в зону отслеживания для момента сохранения
  • git commit -m 'сообщение сохранено' - момент сохранения
  • git status - это проверка состояния файла
  • gitk - это открытие программы для просмотра репозитория
  • git branch - список веток. git branch name cоздаться ветка с именем name
  • git checkout - переходы по веткам и по коммитам
  • git merge - это команда слияние веток
